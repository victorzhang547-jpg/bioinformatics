# 1. Lecture 1 Notes

## 1.1 What is Bioinformatics?

Bioinformatics is an interdisciplinary field that combines:

- Biology
- Computer Science
- Mathematics
- Statistics

Its core goal is:

> To use computational methods to analyze biological data.

Modern biology produces massive amounts of data (e.g., DNA sequences, RNA expression data, protein structures).  
Bioinformatics provides tools and algorithms to process and interpret these data.

---

## 1.2 What is an Algorithm?

An **algorithm** is a step-by-step procedure used to solve a problem.

Characteristics of an algorithm:

- Finite (must terminate)
- Clear and unambiguous
- Has input and output
- Reproducible

Example in bioinformatics:

- Sequence alignment algorithm (e.g., matching DNA sequences)
- Sorting gene expression values
- Searching for motifs in DNA

In simple terms:

> An algorithm tells the computer *how to compute*.

---

## 1.3 What is a Model?

A **model** is a simplified representation of a real-world system.

It helps us:

- Understand biological mechanisms
- Make predictions
- Simulate biological processes

Examples:

- Evolutionary models
- Gene regulation models
- Protein structure models

In simple terms:

> A model describes *how the biological system behaves*.

---

## 1.4 Difference Between Algorithm and Model

| Algorithm | Model |
|:-----------:|:--------:|
| A procedure | A representation |
| Focuses on *computation* | Focuses on *explanation* |
| Solves a problem | Describes a system |
| Step-by-step rules | Mathematical or conceptual framework |

Relationship:

- A **model** describes biological phenomena.
- An **algorithm** is often used to compute or implement a model.

Example:

If we build a model of DNA mutation,  
we need an algorithm to simulate mutation processes.
